```env
# Данные отправителя (обязательно заполните!)

CDEK_SENDER_NAME=Ваше имя
CDEK_SENDER_PHONE=+79991234567

# Тариф по умолчанию
CDEK_DEFAULT_TARIFF_NAME=Магистральный экспресс склад-склад

# Тип заказа:
# 1 = интернет-магазин (рекомендуется)
# 2 = доставка
CDEK_ORDER_TYPE=1

# ПВЗ отправителя (опционально)
# CDEK_SENDER_PVZ=IZHV...
```

## Важно:

1. **CDEK_SENDER_NAME** - укажите ваше реальное имя или название компании
2. **CDEK_SENDER_PHONE** - укажите ваш реальный телефон в формате +79991234567
3. Если используете тестовый сервер, измените `CDEK_BASE_URL` на `https://api.edu.cdek.ru`
4. **CDEK_ORDER_TYPE=1** - режим `интернет-магазин` (если нужно, можно переключить на `2` для режима `доставка`)
5. При создании отправки бот сначала использует `CDEK_ORDER_TYPE`, а если тарифы не найдены, автоматически пробует второй тип.
