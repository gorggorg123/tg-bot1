# Улучшения AI-модуля

## Обзор изменений

В рамках оптимизации AI-модуля были внесены следующие улучшения:

### 1. Метрики AI (AIMetrics)

Добавлен класс `AIMetrics` для отслеживания:
- Количество запросов (успешных/неудачных)
- Использование токенов (input/output)
- Примерная стоимость запросов в USD
- Среднее время ответа (latency)
- Количество ретраев
- Типы ошибок

```python
from botapp.ai_client import get_ai_metrics

metrics = get_ai_metrics()
print(metrics.as_dict())
# {
#     "total_requests": 150,
#     "successful_requests": 145,
#     "failed_requests": 5,
#     "success_rate": "96.7%",
#     "total_input_tokens": 45000,
#     "total_output_tokens": 12000,
#     "avg_latency_ms": 850.5,
#     "estimated_cost_usd": 0.0234,
#     ...
# }
```

### 2. Экспоненциальный Backoff с Jitter

Улучшена логика retry при ошибках API:
- Начальная задержка: 0.5 секунды
- Множитель: 2x
- Максимальная задержка: 10 секунд
- Jitter: ±30% для предотвращения thundering herd

```
Attempt 1: fail → wait ~0.5s
Attempt 2: fail → wait ~1.0s
Attempt 3: fail → wait ~2.0s
Attempt 4: fail → raise error
```

### 3. Кэширование ответов

Добавлено in-memory кэширование похожих запросов:
- TTL: 5 минут
- Максимум: 100 записей
- Ключ: хэш от system+user+model

Позволяет экономить токены при повторных запросах.

### 4. Улучшенная обработка ошибок

Добавлены специфичные исключения:
- `AIClientError` — базовая ошибка
- `AIRateLimitError` — превышен rate limit
- `AIQuotaExceededError` — превышена квота

Классификация ошибок для метрик:
- `rate_limit` — 429
- `auth_error` — 401
- `server_error` — 500-504
- `timeout` — таймауты
- `connect_error` — ошибки соединения

### 5. Исправление chats_ai.py

Исправлена критическая ошибка несовместимости сигнатур.
Функция `suggest_chat_reply` теперь корректно вызывает `generate_chat_reply`.

### 6. Обновление reviews_ai.py

Модуль теперь использует настоящий AI вместо заглушки.
При отсутствии API ключа — возвращает шаблонный ответ.

```

Запросы:
├ Всего: 150

├ Ошибок: 5
└ Success rate: 96.7%

Токены:
├ Input: 45,000
├ Output: 12,000
└ Всего: 57,000

Производительность:
├ Avg latency: 850ms
├ Retries: 3
└ Стоимость: $0.0234
```

### /ai_reset
Сбрасывает статистику AI (только для админов).

## Конфигурация

Новые переменные окружения в `.env`:

```env
# OpenAI Configuration
OPENAI_API_KEY=sk-proj-xxx          # API ключ (обязательно)
OPENAI_MODEL=gpt-4o-mini            # Модель (по умолчанию gpt-4o-mini)
OPENAI_TIMEOUT_S=35                 # Таймаут в секундах
OPENAI_MAX_RETRIES=3                # Макс. количество ретраев
```

## Рекомендации

1. **Мониторинг** — регулярно проверяйте `/ai_stats` для контроля расходов
2. **Кэширование** — похожие запросы автоматически кэшируются на 5 минут
3. **Rate limits** — при превышении лимитов бот автоматически делает retry
4. **Fallback** — при недоступности API возвращаются шаблонные ответы
