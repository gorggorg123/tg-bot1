# Интеграция стиля ITOM в ответы OpenAI

Справочник с типовыми ответами и правилами стиля хранится в `data/itom_qna_digest.txt`.
Его содержимое добавляется в системные подсказки при генерации ответов:

- `generate_answer_for_question` — добавляет текст справочника в system prompt для ответов на вопросы покупателей.
- `generate_review_reply` — использует тот же справочник, чтобы черновики для отзывов соответствовали стилю бренда.

По умолчанию файл читается из `data/itom_qna_digest.txt`. Путь можно переопределить
переменной окружения `ITOM_QNA_DIGEST_PATH`, чтобы обновлять или экспериментировать со справочниками
без изменения кода. Файл загружается лениво и кэшируется внутри `botapp.ai_client`.

Чтобы обновить справочник, отредактируйте текст или укажите новый путь в `ITOM_QNA_DIGEST_PATH`.
После изменения файла перезапустите процессы бота, чтобы кэш обновился.
