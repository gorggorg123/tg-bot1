# 🚀 Как запустить бота

## Быстрый старт

### 1. Установите Python 3.11+

Скачайте и установите Python с [python.org](https://www.python.org/downloads/)

Или используйте скрипт:
```bash
INSTALL_PYTHON.bat
```

### 2. Установите зависимости

```bash
pip install -r requirements.txt
```

### 3. Настройте .env файл

Создайте файл `.env` в корне проекта (пример в `ENV_EXAMPLE.txt`):

```env
# Telegram Bot
TG_BOT_TOKEN=your_telegram_bot_token_here

# Ozon API (обязательно!)
OZON_CLIENT_ID=your_ozon_client_id
OZON_API_KEY=your_ozon_api_key

# OpenAI (опционально, для AI функций)
OPENAI_API_KEY=your_openai_key

# Admin (ваш Telegram ID)
ADMIN_IDS=123456789
```

### 4. Запустите бота

```bash
python run_local.py
```

**Вот и всё!** 🎉

Все логи будут отображаться прямо в терминале для удобной отладки.

## Остановка бота

Просто нажмите `Ctrl+C` в терминале

## Получение Telegram ID

1. Напишите боту [@userinfobot](https://t.me/userinfobot)
2. Скопируйте ваш ID
3. Добавьте в `.env` как `ADMIN_IDS`

## Получение Ozon API ключей

1. Зайдите в [Личный кабинет продавца Ozon](https://seller.ozon.ru)
2. Настройки → API ключи
3. Создайте новый ключ с правами на чтение отзывов, вопросов, заказов
4. Скопируйте Client ID и API Key в `.env`

## Устранение проблем

### Бот не запускается

- Проверьте что установлен Python 3.11+: `python --version`
- Проверьте что установлены зависимости: `pip install -r requirements.txt`
- Проверьте что файл `.env` создан и заполнен

### Ошибки с SSL/сертификатами

Если видите ошибку `SSL: WRONG_VERSION_NUMBER` или `ClientConnectorSSLError`:

**Решение 1 (рекомендуется для разработки):**
Добавьте в `.env`:
```env
TG_DISABLE_SSL_VERIFY=1
```

**Решение 2 (для production):**
Добавьте в `.env`:
```env
TG_ENABLE_SSL_VERIFY=1
TG_SSL_CA_FILE=/path/to/ca.pem
```

⚠️ **Внимание:** Отключение SSL небезопасно! Используйте только для разработки или за корпоративным прокси.

### Нет прав на отправку ответов

Убедитесь что API ключ Ozon имеет права:
- Чтение и запись отзывов
- Чтение и запись чатов
- Чтение заказов
